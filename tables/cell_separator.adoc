== Change cell separator

The `separator` block attribute allow you to change separator.

// tag::change_separator[]

.Default
|===
|1|2|3
|===

.Change separator to `$`
[separator="$"]
|===
$1$2$3
|===

// end::change_separator[]

== Escape bar in a Table

There are three ways to display bar (`|`) in a table.

// tag::escape_bar[]

.Escape (`\|`)
|===
|\|
|===

.Change separator
[separator="$"]
|===
$|
|===

.Use `\{vbar}` attribute
|===
|{vbar}
|===
