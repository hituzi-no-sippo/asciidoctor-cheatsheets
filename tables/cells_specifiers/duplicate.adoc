Duplicate Cells^]

Cell duplicate allow you to define consecutive same cells in a table.

Syntax is
`<NUMBER_OF_CELLS>*<HORIZONTAL_ALIGN><VERTICAL_ALIGN><CONTENT_FORMAT>`.
For Example `4*^.^l`.

[CAUTION]
====
Only one of the duplicate (`*`) and span (`+`) can be used.
====
