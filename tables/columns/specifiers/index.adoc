Specify the number of columns with the cols attribute - Add Columns to a Table^]

Use column specifier on `cols` block attribute.

.The column specifier defines elements
Width::
* `3`
* `30%`
Align::
* `^`
* `.^`
Format Content::
* `s`
* `a`
Number of Columns::
* `2*`
* `5*`
